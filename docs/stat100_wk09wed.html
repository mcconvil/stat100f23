Question: How do sampling distributions help us quantify uncertainty?

Statistical Inference

Goal: Draw conclusions about the population based on a sample.

Main Flavors:

  • Estimating numerical quantities.
  • Testing conjectures.

Example: Does Extrasensory Perception (ESP) exist?

Bem and Honorton conducted extrasensory perception studies:

  • A ‚Äúsender‚Äù randomly chooses an object out of 4 possible objects and sends that information to a ‚Äúreceiver‚Äù.
  • The ‚Äúreceiver‚Äù is then given a set of 4 possible objects and they must decide which one most resembles the object sent to them.

Out of 329 trials, the ‚Äúreceivers‚Äù correctly identified the object 106 times.

ESP Example

Let‚Äôs consider the following questions:

  1. If ESP does not exist and the ‚Äúreceivers‚Äù are guessing, how often would we expect them to be correct?

  2. For each sample (set of 329 trials), do we expect the proportion of correct guesses to be equal? Why or why not?

  3. Is it possible to randomly guess correctly 106 out of 329 times (i.e., 32% of the time)?

  4. How unusual is it to guess correctly 106 out of 329 times if ESP doesn‚Äôt exist?

To help us answer d., we need a sampling distribution for the sample proportion where we assume the ‚Äúreceivers‚Äù were purely guessing!

Sampling Distribution of a Statistic

Steps for (Approximate) Distribution:

  1. Decide on a sample size, \(n\).

  2. Randomly select a sample of size \(n\) from the population.

  3. Compute the sample statistic.

  4. Put the sample back in.

  5. Repeat Steps (2) - (4) many (1000+) times.

Hypothesis Testing

Big Idea:

  • Make an assumption about the population parameter.

  • Generate a sampling distribution for a test statistic based on that assumption.

    • Called a null distribution

  • See if the test statistic based on the observed sample aligns with the generated sampling distribution or not.

  • If it does, then we didn‚Äôt learn much.

    • (Didn‚Äôt prove the parameter equals the assumed value but it is still plausible)

  • If it doesn‚Äôt, then we have evidence that our assumption about the parameter was wrong.

ESP Example

Big Idea:

  • Make an assumption about the population parameter.
    • Ex: ESP doesn‚Äôt exist. p, probability of guessing correctly, equals 0.25.
  • Generate a sampling distribution for a test statistic based on that assumption.
    • Called a null distribution

ESP Example

Big Idea:

  • See if the test statistic based on the observed sample aligns with the generated sampling distribution or not.
    • Ex: It is in the center-ish of the distribution. It isn‚Äôt an unusual value.
  • If it does, then we didn‚Äôt learn much. (Didn‚Äôt prove the parameter equals the assumed value but it is still plausible)
    • It is still possible that ESP doesn‚Äôt exist.

ESP Example

Big Idea:

  • See if the test statistic based on the observed sample aligns with the generated sampling distribution or not.
    • It is far in the tails of the distribution. It is an unusual value.
  • If it doesn‚Äôt, then we have evidence that our assumption about the parameter was wrong.
    • We have evidence ESP exists.
